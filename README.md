# Root Quest - Juego en Lenguaje Jack

## Descripción del Juego
**Root Quest** es un juego desarrollado en lenguaje Jack donde controlas una raíz que debe crecer hacia abajo en busca de agua, evitando obstáculos y piedras en su camino.

### Características Principales
- Control intuitivo con flechas del teclado
- Generación de obstáculos estratégicos
- Sistema de puntuación progresivo
- Mecánica de crecimiento realista de raíces

## Tecnologías Utilizadas
- **Lenguaje**: Jack
- **Plataforma**: Nand2Tetris
- **Herramientas**: JackCompiler, VM Emulator

##  Controles del Juego
- **Flecha Izquierda (←)**: Mover raíz hacia la izquierda
- **Flecha Derecha (→)**: Mover raíz hacia la derecha
- **Flecha Arriba (↑)**: Centrar la raíz
- **Movimiento Automático**: La raíz crece hacia abajo constantemente

##  Estructura del Proyecto

RootQuest/  
├── **Main.jack** — Punto de entrada del programa  
├── **Game.jack** — Lógica principal del juego  
├── **Root.jack** — Control y comportamiento de la raíz del jugador  


##  Instrucciones de Ejecución
1. Compilar los archivos .jack usando JackCompiler
2. Ejecutar en VM Emulator de Nand2Tetris
3. Usar las flechas del teclado para controlar la raíz
4. Esquivar obstáculos y llegar al agua para sumar puntos

## Características Técnicas
- **Pantalla**: 512x256 píxeles
- **Obstáculos**: obstáculos distribuidos estratégicamente
- **Puntuación**: +10 puntos por cada vez que se alcanza el agua
- **Colisiones**: Detección precisa con obstáculos y bordes
